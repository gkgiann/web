Questão 03:
Acredito que é adequado usar o css grid quando existem algumas linhas e colunas na página, onde cada parte do layout vai ter o seu espaço pra ocupar.
Por exemplo o site do intagram que tem uma bar no lado esquerdo com atalhos pra acessar as mensagens, reels, pesquisa, notificações, etc. No centro tem o feed com os posts e em cima disso os storys.
No lado direito tem sugestões de pessoas para seguir e outros detalhes de copyright e termos de uso, etc.
Então pra esse caso é mais fácil fazer esse layout com css grid do que flexbox, e melhor ainda creio que seria unir eles, usando ambos juntos o "poder" é dobrado.
